# 哈佛谈判术教练 (harvard-negotiation)

一个基于哈佛谈判项目《Getting to Yes》原则式谈判方法论的谈判教练插件。

## 它能做什么

当你面临任何谈判时，它会：

1. **苏格拉底式分步提问**——一次问几个关键问题，帮你从"立场"挖到真正的"利益"，量化你的目标、底线和退路（BATNA）。
2. **一步步引导你制定策略**——摸清对方、找出双方可成交的重叠区、用客观标准支撑要价、设计让步节奏、写出沟通话术、预演并反制对方的强硬手段。

## 适用场景

买房砍价、租房谈租金、二手交易、离婚财产/抚养权协议、离职补偿/裁员谈判、加薪升职、offer 谈薪、商业合同与合作分成、供应商客户议价等。

## 怎么用

安装后，直接说出你的处境即可触发，例如：

- "我下周要和公司谈离职补偿，帮我准备一下"
- "看中一套房，挂牌 500 万，怎么砍价"
- "房东要涨租 15%，我该怎么谈"

它会先问你几个问题，再带你一步步做出策略。

## 结构

- `skills/harvard-negotiation/SKILL.md` — 主引导流程（六步）

- `references/scenarios.md` — 各场景专属筹码与话术

## 说明

本插件提供谈判策略与话术，不构成正式法律或财务意见。涉及离婚、补偿、合同等有法律后果的事项，请咨询执业律师或专业人士。
