# 强硬战术、反制与检查清单

## 谈判柔术（应对强硬的总原则）
对方进攻时，不反击、也不让步，而是**把力道引向问题本身**：
- 对方攻击你的立场 → 不为立场辩护，**邀请他批评、追问"你这么定的依据是什么"**。
- 对方攻击你这个人 → 把攻击**重新表述成对问题的关注**，不接情绪招。
- 多用**提问代替陈述**，问完保持**沉默**——沉默会逼对方填补、暴露更多信息。

## 常见强硬战术与反制

| 战术 | 表现 | 反制 |
|---|---|---|
| 极端开价/狮子大开口 | 先抛离谱数字设锚 | 不被锚住，拉回客观标准，自己亮出有依据的反锚 |
| 最后通牒 / "就这价，不要拉倒" | 施加时间或绝对压力 | 不当场就范；测试真假——回到 BATNA："那我可能得选别的了"，常能让对方松动 |
| 红脸白脸 | 一个凶一个善 | 点破套路或坚持只跟能拍板的人按标准谈 |
| 挤牙膏 / 得寸进尺 | 谈好后又加要求 | 每次新要求都重新索取对等回报，或重开整体一揽子 |
| 装可怜 / 诉诸情分 | "你就帮帮忙" | 共情但回到客观标准与自身底线，情绪归情绪、数归数 |
| 虚假权限 | "我做不了主，得请示" | 提前确认谁拍板，尽量直接对接决策者 |
| 故意拖延施压 | 拖到你着急 | 提前设定自己的时间表，强化 BATNA，不暴露急切 |
| 既成事实 / 文件陷阱 | "标准合同，都这么签" | 没有"不能改"的条款；逐条审，把口头承诺写进文本 |

## 认知偏误（自检，别被它们坑）
- **锚定效应**：第一个数字会拽住你的判断——上桌前自己先想好目标与依据。
- **固定蛋糕错觉**：误以为对方所得即你所失，错失双赢交换。
- **沉没成本**：因"已经投入很多"而硬谈下去——决策只看未来。
- **过度自信 / 确认偏误**：高估自己 BATNA、只听支持自己的信息——主动找反证。
- **损失厌恶**：把同一方案包装成"避免损失"比"获得收益"更打动对方。
- **互惠原则**：先给一个小让步/善意，对方更可能回报——但警惕被对方用来套你。

## 上桌前检查清单
1. 我一句话的目标和**最差可接受线（保留价）**是什么？
2. 我的 **BATNA** 是什么？能不能在上桌前先把它做得更强？
3. 对方的利益、压力、**BATNA**、谁拍板，我清楚吗？
4. 我有哪些**客观标准/依据**支撑要价？
5. 我准备了几个**做大蛋糕的交换方案**？
6. 我的**首报价（锚）**和**让步地图**（先让什么、换什么、绝不让什么）定了吗？
7. 开场、要价、被拒后的**话术**演练过吗？
8. 哪些事项必须**落成书面/合同**？需要律师把关吗？

## 控场提醒
- 会问 > 会说；倾听是最被低估的谈判技能。
- 善用沉默，不急于填补空白。
- 任何重大让步都可以说"我需要想一下"，绝不在压力下当场拍板。
- 始终对事不对人：可以对问题强硬，对人要软。
