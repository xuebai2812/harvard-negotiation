# 原则式谈判 · 进阶框架

源自哈佛谈判项目（Roger Fisher & William Ury,《Getting to Yes》）。这是方法论参考，引导用户时按需调用，不要照本宣科。

## 四大原则

1. **把人和问题分开（Separate people from the problem）**
   情绪、误解、面子是真实存在的"人"的问题，要单独处理：先承认对方情绪、主动倾听、用"我"陈述（"我担心…"而非"你总是…"）。处理完情绪，再回到实质问题。两人并肩看问题，而不是隔桌互相瞪。

2. **聚焦利益，而非立场（Focus on interests, not positions）**
   立场是要求（"租金不能涨"），利益是动机（"我怕负担不起、怕被随意赶走"）。挖利益的提问："为什么这对你重要？""如果不是这样，你担心会发生什么？"双方立场对立时，底层利益往往可以同时满足。经典案例：两人抢一个橙子，一个要果肉榨汁、一个要果皮烤蛋糕——立场冲突，利益互补。

3. **创造对双方有利的选项（Invent options for mutual gain）**
   先发散后收敛：把"想方案"和"做决定"分开。寻找**低我成本、高对方价值**（及反向）的交换项，做大蛋糕。避免过早评判、避免认定"只有一块固定蛋糕"。

4. **坚持客观标准（Insist on objective criteria）**
   让结果取决于独立标准而非意志比拼：市场价、专家评估、法律法规、行业惯例、成本核算、先例。话术："我们能不能用一个双方都认的标准来定？这个数你是按什么算出来的？"

## 立场-利益冰山模型（哈佛训练营核心图解）

把每一方都想成一座冰山，两座冰山隔着谈判桌对望：

```
      我方                              对方
       /\            水上              /\
      /立\   ← 说出口、常针锋相对 →    /立\
─────/场 \──────── 水面 ────────────/场 \─────
    / 利益 \   ← 没说出口、为什么 →  / 利益 \
   /(为什么)\                      /(为什么)\
  /  深层需求 \  ←—— 常有交集 ——→ /  深层需求 \
 / 恐惧·价值·  \                  / 恐惧·价值· \
/  安全感       \                /   安全感     \
      └──────────►  成交区  ◄──────────┘
                  共同利益 / ZOPA
```

- **水面以上的尖（立场）**：双方说出口的要求（"降 50 万""补偿 N+3"），往往直接冲突——在这里硬碰硬只会僵住。
- **水面以下（利益 → 深层需求）**：真正的动机、恐惧、价值、安全感。冰山的体积都在水下，**越往下挖，两座冰山越可能在某处交叠**，那块交叠就是成交区（ZOPA）。
- **每座冰山中间的"梯子（ladder）"**：从立场一级一级往下问"为什么"，从立场 → 利益 → 深层需求逐层下降。

**怎么在对话里用这张图**：当用户死咬一个立场、或与对方僵住时，主动把这张图（口头或画出来）讲给用户听，然后带他做两件事——①顺着自己的梯子往下问"我为什么非要这个"；②推测对方的梯子，问"他水面下真正怕的、要的是什么"。两条梯子下降后再找交叠点，僵局往往就化开了。这是第 1、2 步引导的底层模型。

## 七要素（系统体检表）

谈判前逐项过一遍：
1. **利益 Interests** — 双方各自真正要的是什么（含隐性的：安全感、地位、关系）。
2. **选项 Options** — 桌面上能创造的所有可能方案。
3. **替代方案 Alternatives / BATNA** — 不达成协议时各自的最佳退路。
4. **正当性 Legitimacy** — 支撑提议的客观标准与依据。
5. **沟通 Communication** — 是否双向倾听、信息是否清晰。
6. **关系 Relationship** — 是否在保护/损害长期关系。
7. **承诺 Commitment** — 最终承诺是否具体、现实、可执行、可落地为书面。

## BATNA：议价力的真正来源

- **BATNA = Best Alternative To a Negotiated Agreement**，谈崩后你最好的那条路。
- 三步用好它：①列出所有替代方案 ②挑出最好的并尽量把它做实做强 ③用它来定你的"保留价"（低于它就该走人）。
- **改善 BATNA 比在桌上耍技巧更有效**。上桌前先去拿第二个 offer、找替代房源、咨询律师——你的退路变好，整盘棋就变了。
- 也要**估对方的 BATNA**：对方退路越差，你越有底气；对方退路很强时，靠关系与创造性方案取胜，而非硬刚。

## ZOPA 与定价三锚

- **保留价（Reservation point）**：你能接受的最差条件，由 BATNA 决定。
- **期望价（Aspiration / target）**：你真正想要、且有依据的高目标。研究显示，**目标定得高（但有理有据）的人最终拿得更多**。
- **ZOPA（Zone of Possible Agreement）**：双方保留价之间的重叠区。有重叠才有得谈；没有重叠，要么扩大蛋糕、要么改善 BATNA、要么不谈。

## 锚定与让步

- **首报价（锚）**：在信息充分时，主动用一个有依据、略高于目标的开价设锚，能拉动整个区间向你倾斜；信息不足或风险高时，可让对方先报。
- **让步原则**：①每次让步都索取对等回报（"我可以…前提是你…"）②让步幅度递减，传递"快到底线"的信号 ③绝不无理由地连续单方让步 ④把让步包装成"为你做的努力"。
- **MESO（多个等值同时报价）**：同时给出 2–3 个对你价值相当的方案让对方选，既显灵活，又能从对方的偏好里读出他真正在乎什么。

## 常见误区
- 把谈判当零和博弈，只盯一个数字砍。
- 不做功课就上桌（不知道自己和对方的 BATNA、没有客观依据）。
- 被情绪带走，对人不对事。
- 不敢要、目标定太低，开口就接近底线。
- 在压力/催促下当场答应——任何重大让步都值得说"我需要想一下"。
