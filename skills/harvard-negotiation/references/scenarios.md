# 场景专属手册

每个场景给出：典型利益、关键 BATNA、可用客观标准、常见可交换筹码、易踩的坑。引导时挑与用户相关的点提问，不要全盘背诵。

---

## 1. 买房砍价

- **你的利益**：总价、首付/贷款节奏、过户时间、维修与家电、风险（产权、学区）。
- **对方利益**：卖家常在乎"快、稳、净到手价"——时间紧、缺钱、怕夜长梦多的卖家最好谈。
- **BATNA**：你的——有没有备选房源、租住能否再等；卖家的——挂了多久、有没有别的买家。挂牌越久、看房越少，卖家 BATNA 越弱。
- **客观标准**：同小区/同户型近 3–6 个月**成交价**（不是挂牌价）、单价、房龄、楼层差价、税费、评估价。
- **可交换筹码**：全款或快速放款、缩短过户周期、承担部分税费、不要某些家具家电以换降价、配合卖家的搬离时间。
- **坑**：暴露急切、爱上某套房失去 BATNA、只砍总价忽略税费/维修等隐性成本。

## 2. 租房谈租金

- **你的利益**：月租、押付方式、租期稳定、维修责任、涨租上限、退租灵活。
- **房东利益**：空置最贵——稳定长租、按时付款、爱护房子的租客对房东价值高。
- **BATNA**：周边同类房源价格与空置量；淡旺季（毕业季房东强、淡季租客强）。
- **客观标准**：周边同小区在租挂牌价、租金指数、空置时长。
- **可交换筹码**：签更长租期、押一付三/押一付六、自己承担小维修、好房客信用——换月租下调或免涨。
- **坑**：只盯月租不谈涨租条款和退租违约；口头承诺不写进合同。

## 3. 离婚协议（财产 / 抚养权）

- **提醒**：这是有法律后果的领域，关键条款务必请执业律师把关；你提供的是策略与沟通思路，不是法律意见。
- **你的利益**：财产分割、孩子抚养与探视、抚养费、稳定与体面收场、降低对孩子的伤害、长期可执行。
- **对方利益**：往往也想要确定性、面子、少纠缠、孩子好。找共同利益（孩子福祉）做支点。
- **BATNA**：诉讼判决的大致预期——了解法律默认结果（你诉讼能拿到什么），它就是你协议的保留线。
- **客观标准**：法律对共同财产/抚养费的规定、当地判例、孩子实际需求清单、双方收入证明。
- **可交换筹码**：财产 vs 抚养权 vs 探视时间 vs 一次性付清 vs 分期，不同事项对双方价值不同，可打包交换。
- **坑**：让情绪主导（对人不对事）、为赌气牺牲实际利益、协议含糊导致日后反复执行难。

## 4. 离职补偿 / 裁员谈判

- **你的利益**：补偿金额（N、N+1、2N…）、在职/离职日期、社保公积金延续、未休年假/奖金/期权、竞业与保密、背调与离职证明措辞、推荐信。
- **公司利益**：合规、快速干净了结、避免劳动仲裁与负面影响、保证项目交接平稳、保护商业机密。
- **BATNA**：你的——下一份 offer 进度、仲裁胜算与时间成本、储蓄能撑多久；公司的——直接辞退的法律风险与赔偿、找人接替的成本。**手上有 offer 或仲裁筹码时议价力最强。**
- **客观标准**：劳动法对经济补偿/赔偿金的规定（如 N+1、2N 的适用情形）、合同条款、同司先例、你的司龄与月薪基数。
- **可交换筹码**：配合交接 / 保密不声张 / 接受协商离职（而非被辞退） ↔ 换更高补偿、延后离职日期、保留期权或奖金、好的离职证明。把"平稳交接"当成你的有价资产去交换。
- **坑**：情绪化裸辞丢掉筹码、不留书面证据、签字前没核算应得总额、忽视竞业条款的后续影响、留下职业 gap。
- **与本项目相关提示**：若用户目标是"补偿最大化 + 延长在职 + 完成项目 + 不留 gap"，引导用"配合完成项目与交接"换"延后离职日期 + 更高补偿"，并把"无缝衔接下家"纳入时间线设计。

## 5. 加薪 / 升职 / offer 谈薪

- **你的利益**：基本工资、奖金、股权、签字费、职级、头衔、远程/弹性、入职时间、晋升时间表。
- **对方利益**：留住人才、控制薪酬带宽与内部公平、快速到岗。
- **BATNA**：你的——其他 offer、现职、市场需求；公司的——其他候选人、岗位紧迫度。**竞品 offer 是最硬的锚。**
- **客观标准**：市场薪资分位数据（同岗位同城同年限）、你的业绩量化成果、公司薪酬带宽、过往晋升标准。
- **可交换筹码**：基本工资 vs 签字费 vs 股权 vs 入职时间 vs 头衔——若现金谈不动，换一次性签字费或更早晋升评估。
- **坑**：先报数字暴露底牌（信息不足时让对方先报）、只谈底薪忽略总包、用"我需要钱"而非"我创造的价值/市场行情"做理由。

## 6. 商业合同 / 合作分成 / 供应商议价

- **你的利益**：价格、付款账期、交付与质量、违约责任、排他性、续约与涨价条款、知识产权归属。
- **对方利益**：现金流、长期订单、可预测性、降低风险。
- **BATNA**：替代供应商/客户的报价与质量；切换成本。
- **客观标准**：市场报价、成本构成、行业标准合同条款、过往合作数据。
- **可交换筹码**：更长合约/更大订单量 ↔ 单价下调；预付/快付 ↔ 折扣；排他 ↔ 价格保护。
- **坑**：只谈单价忽略账期与违约条款、口头承诺不入合同、忽视续约时的涨价/锁定风险。
